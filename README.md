
# Проекты:

- **[03_btl_agencies_collector](https://github.com/KirPresnyakov/03_btl_agencies_collector.git)** — сборщик данных BTL и маркетинговых агентств
- **[04_Adding-a-new-source-to-DWH](https://github.com/KirPresnyakov/04_Adding-a-new-source-to-DWH.git)** — добавление нового источника в DWH, загрузка, создание витрины

- **[09_Cloud_Technologies](https://github.com/kirpresnyakov/09_Cloud_Technologies.git))** — разработка сервисов на платформе Yandex Cloud,  реализующих ETL-процесс загрузки данных в слои DWH. PostgreSQL/Kafka/Valkey/Kubernetes/Helm.
