
# Проекты:

- **[04_Adding_new_source_to_DWH](https://github.com/KirPresnyakov/04_Adding_new_source_to_DWH.git)** — добавление нового источника в DWH, загрузка, создание витрины. Стек: PostgreSQL.
- **[05_DWH_multi_sources](https://github.com/kirpresnyakov/05_DWH_multi_sources.git)** — интеграция нескольких источников в DWH. Стек: PostgreSQL,MongoDB, Apache Airflow.
- **[06_Analytical_database](https://github.com/kirpresnyakov/06_Analytical-database.git)** — создание DWH по модели Data Vault в Vertica (загрузка данных, построение аналитической витрины). Стек: Vertica, S3, Apache Airflow.
- **[07_Data_Lake](https://github.com/kirpresnyakov/07_Data_Lake.git))** — разработка Data Lake. Hadoop, Apache Spark, Apache Airflow.
- **[09_Cloud_Technologies](https://github.com/kirpresnyakov/09_Cloud_Technologies.git))** — разработка сервисов на  облачной платформе,  реализующих ETL-процесс загрузки данных в слои DWH. Стек: Yandex Cloud, PostgreSQL, Apache Kafka, Valkey, Kubernetes, Helm.
